Assignments for ENGI 1006 - Introduction to Computing (Spring 2020)

This is a summary of the homework assignments for the "ENGI 1006 - Introduction to Computing for Engineers and Applied Scientists" class. The first half of the semester included a combination of theory and practice assignments, while the second half was more focused on applications.

  • Homework 1A
    • Theory assignment on pseudocode, basic algorithms, binary numbers manupulation and truth tables.
  • Homework 1B
    • Design of a simple program with focus on structure and functions use.
    • Functions to execute different manipulations on lists.
  • Homework 2A
    • Theory assignment on sorting algorithms and time complexity.
  • Homework 2B
    • Physics simulation of a projectile with and without air resistance. Included a graph describing the projectile motion.
    • First steps towards building a machine learning application to analyze data about breast cancer. Implementation of basic utility functions for the dataset.
  • Homework 3A
    • Writing pseudocode to implement set-style operations with lists.
    • Comparing the time complexity of doing these operations with lists or dictionaries.
    • Implementation of recursive functions using python code. Included exponential functions, permutations and palindrome detection.
  • Homework 3B
    • Writing a python module to solve the vertical percolation problem. Used numpy.
    • Additions to the machine learning application. Expand utility functions and implement functions to provide statistics on the dataset.
  • Homework 4
    • Assignment on Object-Oriented Programming (OOP).
    • Modelling a class with a teacher, students and assignments.
    • The homework focused on being able to read and understand an object-oriented codebase and interpreting error messages to complete the library.
  • Midterm
    • Writing an application that takes as an input 3 PPM images and outputs an image with unwanted objects removed.
    • Also includes a function to convert an image to black and white, and a function to flip an image horizontally.
  • Homework 5
    • Completing the machine learning library using multiple python libraries, like numpy, sklearn, pandas, and seaborn.
    • This included providing statistics on the dataset, plotting information about the data, and using scikit-learn to implement the K Nearest Neighbors and Support Vector Machine classifier algorithms.
    • We also implemented the KNN algorithm ourselves, and verifying the accuracy of each algorithm.
  • Final
    • You are seeing it rigth now!
    • Using Flask and HTML to create a basic personal webpage.
    • Also included an introduction to Git and GitHub.

My favorite assignment this semester was Homework 5. I enjoyed seeing how the work of the last months resulted in a functioning machine learning application. The assignment combined many libraries that I found to be very useful. I also liked implementing the KNN algorithm by ourselves and seeing that it had over 90% accuracy.
